Librarinth-Web-Client by Thaina

C# Template for Web Browser Plugin

This project used for Open Source of my NPAPI plugin C# project This project based on C# to create web browser plugin on Firefox and Chrome, for the purpose of create individual plugin. And the real goal to be based of rich internet applications framework which used .NET and Mono as Runtime Environment
